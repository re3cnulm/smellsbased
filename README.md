## Setup

1. Create a GitHub repo at `https://github.com/new`
2. Name it `smellsbased`
3. Clone locally or push from here:
   ```bash
   git remote add origin https://github.com/re3cnulm/smellsbased.git
   git branch -M main
   git push -u origin main
   ```

4. Connect to Vercel:
   - Go to https://vercel.com
   - Import from GitHub (select `smellsbased` repo)
   - Add custom domain: `smellsbased.com`
   - Deploy
